# 920. [Комнаты для Переговоров (Meeting Rooms)](https://leetcode.com/problems/meeting-rooms/description/)

`Easy`

**Описание**

> (0,8), (8,10) не конфликтуют в 8

Input: intervals = [(0,30),(5,10),(15,20)]
Output: false
