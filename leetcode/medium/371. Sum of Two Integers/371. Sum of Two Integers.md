# 371. [Сумма Двух Целых Чисел (Sum of Two Integers)](https://leetcode.com/problems/sum-of-two-integers/description/)

`Medium`

Даны два целых числа `a` и `b`, верните *сумму двух целых чисел*, не используя операторы `+` и `-`.

**Пример 1:**
```
Input: a = 1, b = 2
Output: 3
```

**Пример 2:**
```
Input: a = 2, b = 3
Output: 5
```

**Ограничения:**

*   -1000 ≤ a, b ≤ 1000
